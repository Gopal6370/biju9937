# Exploratory Data Analysis (EDA) & Business Intelligence

## 📌 Project Overview

This project was completed as part of the **Data Analytics Internship Program at ApexPlanet Software Pvt. Ltd.**

The objective of this project is to perform **Exploratory Data Analysis (EDA)** and **Business Intelligence Analysis** on customer sales data using Python, SQL, and data visualization techniques. The analysis helps identify sales trends, customer behavior patterns, product performance, and key business insights for data-driven decision-making.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing.
* Analyze sales data using Exploratory Data Analysis (EDA).
* Generate meaningful visualizations and insights.
* Create a SQLite database for business analysis.
* Execute SQL queries to answer business questions.
* Design a Business Intelligence Dashboard Mockup.
* Extract actionable insights from customer sales data.
